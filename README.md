# 📱 SwiftUI Card UI - Ejemplo de interfaz

![SwiftUI](https://img.shields.io/badge/SwiftUI-iOS-blue)

Pequeño proyecto desarrollado con **SwiftUI** para practicar la construcción de interfaces utilizando stacks y estilos básicos.

## Qué incluye

- Uso de VStack, HStack y ZStack
- Imagen con bordes redondeados
- Sistema de estrellas con SF Symbols
- Diseño tipo tarjeta con sombra

## Conceptos practicados

- Composición de vistas
- Layout y spacing
- Modificadores de SwiftUI

## Notas

Proyecto realizado con fines de aprendizaje.
